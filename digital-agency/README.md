# 🌊 Next Wave - Digital Agency Website

Next Wave is a modern and responsive digital agency website built to showcase professional services like web development, app development, UI/UX design, and digital marketing.

## 🚀 Features

* Responsive and mobile-friendly design
* Clean and modern UI
* Services section with detailed offerings
* Portfolio/project showcase
* Contact form interface
* Smooth navigation and user experience

## 💻 Technologies Used

* HTML5
* CSS3
* JavaScript

## 🎯 Purpose

This project is designed for learning, portfolio showcase, and as a starter template for digital agencies or freelancers.
